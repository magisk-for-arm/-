高级编码转换工具

本项目是一个基于 PyQt5 的图形化批量编码转换工具，支持 UTF-8 和 UTF-16LE 之间的互相转换，适用于需要批量转换文本文件编码的场景。
支持自动检测源文件编码、批量拖放导入、BOM 设置、主题切换、备份保护等多种实用功能。
功能特性

    双向编码转换：支持 UTF-8 与 UTF-16LE 互转，灵活满足不同需求。
    批量处理：可添加多个文件，或直接拖放文件夹，一键批量转换。
    自动编码检测：可自动识别源文件编码，提升兼容性。
    BOM 支持：可选择是否添加字节顺序标记（BOM）。
    自定义输出：可选择覆盖原始文件或另存为新目录，避免数据丢失。
    备份功能：支持自动为原文件创建 .bak 备份文件。
    主题切换：内置深色/浅色主题，界面美观。
    跨平台：兼容 Windows、Linux、macOS。


注意事项

    本工具主要适用于文本文件的编码转换，不适用于二进制文件。
    转换时请注意备份重要文件，防止数据丢失。
    若遇到编码无法识别或转换失败，建议手动指定源编码重试。

依赖项

    PyQt5
    chardet
